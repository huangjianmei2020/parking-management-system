
# 智能停车场管理系统

基于 **eladmin** 开源框架二次开发的智能停车场管理系统，采用 **Spring Boot 2.6.4 + Redis** 技术栈，实现了车位管理、车辆出入场、在线预约、AI车牌识别等核心功能。

---

## 🚀 技术栈

| 技术 | 说明 |
|:---|:---|
| **后端框架** | Spring Boot 2.6.4 |
| **核心存储** | Redis（全量数据存储） |
| **辅助数据库** | MySQL 8.0 |
| **ORM框架** | MyBatis-Plus |
| **权限控制** | Spring Security + JWT |
| **AI识别** | 百度AI车牌识别API |
| **接口文档** | Swagger（Knife4j） |
| **构建工具** | Maven |
| **版本控制** | Git |

## ✨ 核心功能

### 1. 车位管理
- 车位的增删改查
- Excel 批量导入/导出
- 车位状态实时更新（空闲/占用/预约/维修）

### 2. 车辆出入场管理
- **AI车牌识别入场**：上传图片自动识别车牌号
- **手动入场**：支持指定车位或系统自动分配
- **车辆出场**：自动计算停车时长，释放车位
- **支付确认**：管理员线下收款后确认支付

### 3. 车位预约管理
- **创建预约**：支持指定车位或系统自动分配
- **取消预约**：限定时间内可取消（默认5分钟）
- **预约到场**：到场后自动转为入场记录
- **超时自动释放**：基于 Redis 过期通知机制，毫秒级自动释放车位

### 4. 预约配置管理
- 预约保留时长（默认30分钟）
- 每日最大预约数（默认200个）
- 取消预约时限（默认5分钟）
- 所有配置实时生效，无需重启服务

### 5. 数据统计
- 今日入场/出场车辆数
- 当前在场车辆数
- 车位占用率
- 按状态、车型、时段等多维度统计
